# Tic Tac Toe Game

## Overview
This is a browser-based Tic Tac Toe game I built using HTML, CSS, and JavaScript, using a fun, pink, neon theme. Two players can take turns selecting cells, and the game automatically detects a winner or a draw. The player can then choose to reset the game and play again!

The project focuses on implementing core game logic using JavaScript while dynamically updating the UI.

---

## Features
- Two-player gameplay (X and O)
- Turn-based system
- Win detection using predefined winning combinations
- Draw detection when all cells are filled
- Reset button to restart the game
- Interactive UI with visual feedback

---

## Technologies Used
- HTML (structure)
- CSS (styling and layout using Grid)
- JavaScript:
  - DOM manipulation
  - Event listeners
  - Arrays for game state
  - Conditional logic
 
---
 
## Challenges
This was my first bigger project and I struglled to understand all of the different options and logic I needed to integrate for a game that seemed so 'simple' from the outside. 
One of the main challenges was creating the win detection logic, I needed to:
- Compare multiple different combinations of positions on the board
- Make sure the game identified a win correctly in every scenario
- and prevent further moves once the game had ended
I managed these challenges but continuously breaking down the issue into manageable portions and focusing on tracking back exactly where the issue was being caused, rather than trying to solve every problem at once. This really helped me to improve my problem-solving skills.
